== Description ==
lightweight, easy to use Javascript injector for radical Web Typography

For more information visit the project`s [GitHub](https://github.com/davatron5000/Lettering.js)

From 1 October 2016 on, this plugin is no longer supported. Please use [Powerpack](http://www.ramoonus.nl/wordpress/powerpack/) instead.

== Installation ==
1. Upload `lettering/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
